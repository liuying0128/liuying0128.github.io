hexo博客搭建(一) 安装和初体验

发表于 2021-07-19

参考官方文档：Hexo

什么是Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装

安装前提

Node.js 和 Git

安装hexo

鼠标右键选择git bash 打开命令行

$ npm install -g hexo-cli (全局安装)或 $ npm install hexo（局部安装）

使用hexo命令

  • npx hexo <command> 注意是npx!

  • 将 Hexo 所在的目录下的 node_modules 添加到环境变量之中即可直接使用 hexo <command>

    ​ （即将有hexo命令的目录添加到环境变量）

建站

$ hexo init <folder> 新建<folder>文件夹为存放网站的文件夹 若不写<folder>则在当前目录下新建

$ cd <folder> 进入文件夹

$ npm install

新建完成后，文件夹目录如下：

本地启动

$ hexo server （简写为$ hexo s）

默认情况下访问网址为 http://localhost:4000/

选项 描述
-p, --port 重设端口
-s, --static 只使用静态文件
-l, --log 启动日记记录，使用覆盖记录格式
